# 🐍 КиберЗмейка

Добро пожаловать в **КиберЗмейку** — консольную игру на Go в стиле классической Snake, но с киберпанк-атмосферой 🤖

## 🎮 Что умеет игра

- 🐍 Управление змейкой в реальном времени.
- 🍏 Еда увеличивает длину змейки и приносит очки.
- 🦠 Вирусы мешают движению: столкновение с ними завершает игру.
- 📈 Уровни растут по мере набора очков.
- 💀 После проигрыша появляется экран `GAME OVER` с возможностью перезапуска.

## 🕹️ Управление

| Клавиша | Действие |
|---|---|
| ⬆️ ⬇️ ⬅️ ➡️ | Движение змейки |
| `W` `A` `S` `D` | Альтернативное управление |
| `Esc` / `Q` | Выход из игры |
| `R` | Перезапуск после поражения |

## ⚙️ Требования

- Go **1.25+**
- Терминал с поддержкой UTF-8
- Поддерживаемая ОС: Linux, macOS, Windows (через совместимый терминал)

## 📦 Установка зависимостей

```bash
go mod tidy
```

Если модуль `go.mod` ещё не создан, сначала инициализируй проект:

```bash
go mod init cyber-snake
go get github.com/nsf/termbox-go
```

## 🛠️ Сборка

```bash
go build -o cyber-snake .
```

## ▶️ Запуск

Можно запустить игру без сборки:

```bash
go run .
```

Или выполнить собранный бинарник:

```bash
./cyber-snake
```

## 🎯 Игровая логика

- 🐍 Змейка движется автоматически по таймеру.
- 🍏 При поедании еды увеличивается счёт.
- 🦠 Каждые несколько очков игра становится опаснее: появляются новые вирусы и растёт уровень.
- 🚫 Столкновение со стеной, телом змейки или вирусом завершает раунд.

## 💡 Подсказки

- Не пытайся развернуться на 180° — такой поворот запрещён.
- Планируй путь заранее, особенно на высоких уровнях.
- Если игра закончилась, нажми `R` для нового раунда или `Q` для выхода.
